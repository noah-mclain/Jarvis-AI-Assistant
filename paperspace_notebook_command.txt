## Paperspace Notebook Command

When creating your Paperspace Gradient Notebook, use the following configuration:

1. Machine Type: P4000, P5000, or P6000 depending on your needs
2. Container: paperspace/gradient-base:pt211-tf215-cudatk120-py311-20240202
3. Command: 

```
PIP_DISABLE_PIP_VERSION_CHECK=1 jupyter lab --allow-root --ip=0.0.0.0 --no-browser --ServerApp.trust_xheaders=True --ServerApp.allow_remote_access=True --ServerApp.allow_origin='*' --ServerApp.allow_credentials=True
```

After your notebook is created:
1. Open a terminal
2. Clone the repository
3. Run the fix script:
```
cd /notebooks
git clone https://github.com/YOUR_USERNAME/Jarvis-AI-Assistant.git
cd Jarvis-AI-Assistant
bash fix_paperspace.sh
```
